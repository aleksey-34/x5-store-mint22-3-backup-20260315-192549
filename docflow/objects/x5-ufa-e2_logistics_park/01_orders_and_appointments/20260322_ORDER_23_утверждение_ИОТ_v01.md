<p><strong>«Об утверждении инструкций по охране труда для работников ИП Исмагилов В.Ш. на объекте "Логистический парк, г. Уфа, Этап 2"».</strong></p>

<p>В соответствии с требованиями Трудового кодекса Российской Федерации (ст. 212, 219), Приказа Минтруда России от 29.10.2021 № 772н «Об утверждении основных требований к порядку разработки и содержанию правил и инструкций по охране труда, разрабатываемых работодателем», в целях обеспечения безопасных условий труда на объекте:</p>

<p><strong>«Логистический парк, расположенный по адресу: Республика Башкортостан, город Уфа, Октябрьский район. Этап 2. Шифр: 10-2/07-2025ПР-1-КЖ3»</strong></p>

<p align="center"><strong>ПРИКАЗЫВАЮ</strong></p>

1. Утвердить и ввести в действие с 22 марта 2026 года следующие инструкции по охране труда:

   | № ИОТ | Наименование инструкции |
   |---|---|
   | ИОТ-001 | Общие требования охраны труда для всех работников |
   | ИОТ-002 | Для монтажников стальных и железобетонных конструкций |
   | ИОТ-003 | Для бетонщиков (укладка, уплотнение, электропрогрев) |
   | ИОТ-004 | Для арматурщиков |
   | ИОТ-005 | Для стропальщиков |
   | ИОТ-006 | Охрана труда при работе на высоте |
   | ИОТ-007 | Электробезопасность (для неэлектротехнического персонала) |
   | ИОТ-008 | Безопасная работа с ручным и электрическим инструментом |
   | ИОТ-009 | Пожарная безопасность на строительной площадке |
   | ИОТ-010 | Погрузочно-разгрузочные работы |

2. Ответственному по ТБ Гречушникову Денису Владимировичу (Приказ № 10/СМР-2026):
   2.1. Ознакомить всех работников организации с инструкциями под роспись в листах ознакомления каждой инструкции в срок до 25.03.2026.
   2.2. Обеспечить наличие комплекта инструкций на рабочих местах (у прораба, в бытовом вагоне) в течение всего периода строительных работ.
   2.3. Проводить повторный инструктаж по утверждённым инструкциям не реже одного раза в 3 (три) месяца.
   2.4. При привлечении новых работников знакомить их с инструкциями до допуска к самостоятельной работе.

3. Начальнику участка Якупову Расиму Рафаиловичу:
   - обеспечить соблюдение требований утверждённых инструкций работниками на объекте.

4. Пересмотр настоящих инструкций — не реже одного раза в 5 лет, или досрочно при изменении законодательства, условий труда, технологии производства работ, либо по результатам расследования несчастного случая.

5. Контроль за исполнением настоящего приказа оставляю за собой.
