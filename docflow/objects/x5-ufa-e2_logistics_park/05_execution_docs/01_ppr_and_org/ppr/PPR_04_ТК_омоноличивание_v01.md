# ПП Р-04: Технологическая карта — Омоноличивание стыков

**Объект:** «Логистический парк», г. Уфа, Этап 2  
**Шифр ПП Р:** 10-2/07-2025/ПР-1-ПП Р  
**Вид работ:** омоноличивание стыков ЖБ конструкций  
**РД:** шифр КЖ3  
**Дата:** март 2026 г.

---

## 1. Область применения

Карта применяется при омоноличивании стыков сборных железобетонных конструкций (колонны, перемычки) бетонной смесью в соответствии с РД КЖ3 и ПП Р.

## 2. Технологическая последовательность работ

| № | Операция | Исполнитель | Примечание |
|---|---|---|---|
| 1 | Проверка правильности установки конструкций перед бетонированием | Прораб | |
| 2 | Установка опалубки (при необходимости) | Плотник / Монтажник | |
| 3 | Зачистка и смачивание поверхности стыка | Бетонщик | |
| 4 | Входной контроль бетонной смеси: марка, осадка конуса, накладная | Прораб | Журнал входного контроля |
| 5 | Укладка бетонной смеси с вибрированием | Бетонщик | |
| 6 | Уход за бетоном (укрытие, полив / электропрогрев) | Прораб | При t < +5°C — электропрогрев (Пр. №20) |
| 7 | Замер температуры бетона (при зимнем бетонировании) | Прораб | Журнал прогрева |
| 8 | Снятие опалубки (при достижении прочности) | Прораб | |
| 9 | АОСР на омоноличивание | Прораб (ИТР) | Форма: `АОСР_шаблон_омоноличивание_v01.md` |

## 3. Требования к качеству бетонной смеси

| Параметр | Требование | НД |
|---|---|---|
| Класс бетона | В____ | РД КЖ3 |
| Водостойкость | W____ | РД КЖ3 |
| Морозостойкость | F____ | РД КЖ3 |
| Осадка конуса (ОК) | ____ – ____ мм | ГОСТ 10181 |
| Температура смеси при укладке | не ниже +5°C (без прогрева) | СП 70.13330.2012 |

## 4. Зимнее бетонирование (при t наружного воздуха ниже +5°C)

В соответствии с Приказом №20/СМР-2026:
- Электропрогрев бетона: ответственный — Мельников (или по назначению);
- Журнал электропрогрева вести с записью температуры каждые 2 часа;
- Набор прочности не менее 70% до замерзания (критическая прочность).

## 5. Состав АОСР на омоноличивание

По каждой захватке оформляется АОСР с приложениями:

| Приложение | Наименование |
|---|---|
| 1 | Паспорт (накладная) на бетонную смесь |
| 2 | Протокол испытания кубиков (R28) — при наличии |
| 3 | Журнал прогрева (при зимнем бетонировании) |

Форма АОСР: `05_execution_docs/02_hidden_and_responsible/hidden_work_acts/АОСР_шаблон_омоноличивание_v01.md`

## 6. Техника безопасности

- СИЗ при работе с бетоном (перчатки, защита глаз);
- При электропрогреве — ограждение опасной зоны, знаки, защита от поражения током;
- Ответственный за электробезопасность: согласно Приказу №05/СМР-2026.
