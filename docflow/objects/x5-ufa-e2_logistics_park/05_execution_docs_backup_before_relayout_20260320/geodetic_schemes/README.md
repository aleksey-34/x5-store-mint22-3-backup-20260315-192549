## Что хранится в этой папке

Исполнительные геодезические схемы (ИГС) — документы, отражающие фактическое положение смонтированных конструкций в пространстве. Оформляются на основе натурных измерений после каждой захватки.

---

## Правила именования файлов

```
ИГС_<тип_конструкции>_<оси>_<дата>_v<NN>.pdf
```

**Примеры:**
- `ИГС_колонны_ряд-А1-А5_20260322_v01.pdf`
- `ИГС_омоноличивание_стык-А2-А3_20260325_v01.pdf`
- `ИГС_разбивка_осей_этаж1_20260318_v01.pdf`

**Дата** — дата выполнения геодезической съёмки (`YYYYMMDD`)  
**v01** — версия, при исправлениях инкрементируется

---

## Состав исполнительной геодезической схемы (ГОСТ Р 51872-2019)

Каждая ИГС должна содержать:

1. **Основная надпись** (штамп) — организация, объект, шифр, масштаб, дата, подписи;
2. **Схема в плане** — фактические координаты смонтированных конструкций с указанием отклонений от проектных осей;
3. **Вертикальный разрез / отметки** — фактические отметки верха колонн/конструкций относительно проектных;
4. **Таблица отклонений** — Δx, Δy, Δz для каждой точки замера, допуск по ГОСТ 21779-82;
5. **Подписи:** геодезист, производитель работ, технадзор Заказчика.

## Порядок оформления и передачи

1. **После разбивки осей** → оформить Акт разбивки осей (шаблон выше) → до начала монтажа;
2. **После монтажа каждой партии колонн** → выполнить геодезическую съёмку → оформить ИГС;
3. **ИГС прикладывается к АОСР** как обязательное приложение (без схемы АОСР не подписывается);
4. **Комплект АОСР + ИГС + паспорта** → сдаётся в Том 3 ИД;
5. **Физические файлы схем** (.pdf / .dwg) хранятся в этой папке.
