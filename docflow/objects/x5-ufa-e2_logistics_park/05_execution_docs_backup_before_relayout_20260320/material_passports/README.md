# Паспортa и сертификаты материалов

Папка для сканов и файлов:

- паспорт колонны
- паспорт бетона
- сертификаты соответствия материалов
- декларации/ТУ, если идут приложениями

Рекомендуемая схема имени файла:

YYYYMMDD__MATPASSPORT__тип_материала__INS-XXX.pdf

Пример:

20260319__MATPASSPORT__бетон_В30__INS-001.pdf
