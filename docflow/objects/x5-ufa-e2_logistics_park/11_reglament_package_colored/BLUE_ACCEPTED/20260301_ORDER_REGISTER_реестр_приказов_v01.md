## Порядок работы с удостоверениями по сканам

1. Сканы удостоверений загрузить в 10_scan_inbox с корректным именем файла.
2. Выполнить ingest-команду для автоматической раскладки.
3. Обновить строки приложений к приказам: N, дата, срок, путь к скану.
4. После проверки перевести файл приказа в подписанную PDF-версию.

## Порядок выпуска финального комплекта

1. Распечатать приказы N 08-11 и наряд-допуск N 12.
2. Подписать у руководителя и ответственных прорабов.
3. Отсканировать подписанные версии в PDF и загрузить в 10_scan_inbox.
4. Выполнить ingest и проверить автоматическую раскладку в папки объекта.
