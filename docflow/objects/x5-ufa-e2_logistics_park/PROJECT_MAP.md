### 1. Сканирование и OCR → `10_scan_inbox/`

```
[Физ. документ]
      │
      ▼
[scan-to-inbox / ручная укладка]
      │
      ▼
10_scan_inbox/YYYYMMDD__DOC_TYPE__SUBJECT__[ID].ext
      │
      ▼ python scripts/scan_archive_docs.py ingest
      │
      ├── AWR   → 05_execution_docs/02_hidden_and_responsible/hidden_work_acts/
      ├── ORDER → 01_orders_and_appointments/
      └── PASSPORT → 02_personnel/employees/<id>/01_identity_and_contract/
           └── нераспознанные → 10_scan_inbox/manual_review/ + review_log.csv
```
