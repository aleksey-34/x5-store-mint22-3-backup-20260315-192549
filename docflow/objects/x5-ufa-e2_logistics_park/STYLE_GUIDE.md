## 2. Правила именования файлов

## 3. Типы разметки документов

### 3.1 Тип А — Официальный распорядительный документ (Приказы, Предписания)

Используется для: Приказов, Нарядов-допусков, официальных уведомлений.  
**Формат:** HTML-в-Markdown (для упрощения печати через браузер/PDF).

```html
<p align="center"><strong>Российская Федерация</strong></p>
<p align="center"><strong>г. Уфа</strong></p>
<p align="center"><strong>Индивидуальный предприниматель<br />Исмагилов Вадим Шакирович</strong></p>

<p align="center"><strong>Приказ № __/СМР-2026</strong></p>

<table style="width: 100%; border-collapse: collapse; margin: 0 0 0.35em 0;">
   <tr>
      <td style="border: none; padding: 0; text-align: left;">г. Уфа</td>
      <td style="border: none; padding: 0; text-align: right;">от __.__.2026г.</td>
   </tr>
</table>

<p><strong>«Наименование приказа».</strong></p>

<p>Преамбула / основание.</p>

<p align="center"><strong>ПРИКАЗЫВАЮ</strong></p>

1. Пункт приказа.
2. Пункт приказа.

<table style="width: 100%; border-collapse: collapse; margin: 0 0 0.25em 0;">
   <tr>
      <td style="border: none; padding: 0; text-align: left; width: 50%;">Индивидуальный предприниматель</td>
      <td style="border: none; padding: 0; text-align: right; width: 50%;">____________________ /Исмагилов В.Ш./</td>
   </tr>
</table>
```

**Таблица ознакомления** (стандартная, 4 колонки):
```html

  <colgroup>

    <th style="border: 1px solid black; padding: 4px 6px; text-align: center;">Ф.И.О. работника, профессия, должность</th>

**Формат:** Чистый Markdown с заголовками (`#`, `##`), таблицами Markdown и полями-подстановками (`__`).

- Дата: __.__.2026

Описание...

## Приложения

|---|---|---|---|
| 1 | ... | | |

Представитель заказчика:  __________________ /______________/
```

## 5. Требования регламента к оформлению (ключевые)

Источник: «Регламент по оформлению и ведению ИД» ООО «Новый элемент» / ООО «СТЗ-Инвест»

- Все документы на **русском языке**; иностранный язык допустим только для наименований оборудования/материалов, не транслируемых на русский.
- Пустые поля заполнять: **«Не применяется»**, **«Отсутствует»**, **«Не требуется»**.
- **Исправления** в документах — только с датой, подписью и расшифровкой; корректор запрещён.
- Хронологический порядок актов в каждой книге.
- В конце комплектования — сквозная нумерация листов (номер в правом нижнем углу).
- Каждая книга: папка-регистратор 70 мм, до 250 листов, реестр в начале.
- На обложке и корешке — **маркировочная этикетка** (форма Приложения 1).
