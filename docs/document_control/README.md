# Document Control Pack

This folder contains a practical documentation system for construction-site onboarding and daily operation control.

## What is included

- folder hierarchy standard for each construction object
- naming and versioning rules
- template pack for orders, personnel files, checklists, and journals
- registry matrix aligned to the provided Form 2 list
- automated scan routing for hidden work acts, passports, and orders
- period zip archive generation for scanned documents

## Intended workflow

1. Create object folder tree from the standard in this pack.
2. Create employee dossiers for every worker and ITR specialist.
3. Fill startup checklist before work admission.
4. Maintain production and HSE journals during execution.
5. Store outgoing responses and monthly control reports.

## Industry and compliance notes

- Template set is aligned with common Russian construction documentation practice.
- Before operational use, verify current revisions and customer-specific requirements for:
  - SP 48.13330 (construction organization)
  - RD-11-05-2007 (general work journal requirements)
  - applicable labor safety and industrial safety regulations for your work scope
